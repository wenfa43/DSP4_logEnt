# Duplicate Slayer Pro v4 (副檔刪檔神器 V4)

**版本:** 26.03.20.logEnT
**作者:** austin\_tseng

## 程式簡介

Duplicate Slayer Pro v4 是一套專門的「重複檔案比對與刪除工具」，專為快速找出硬碟中重複的文件、圖片與影片所設計。
程式具備即時的多格式檔案與多頁面預覽功能，讓您在進行整理或刪除前能先確認檔案內容，以避免誤刪重要資料。

### v4 版本升級重點：

* **全面升級 PDF 預覽核心**：改用高效能原生套件 (PyMuPDF)，解析速度大幅度提升！
* **滾動多頁預覽**：點選 PDF 或文字文件時，右側預覽窗格支援**滑鼠滾輪**往下捲動瀏覽。
* **內建中英雙語切換**：介面上方提供「Language: EN / 語言: 繁中」即時切換按鈕。

### 主要功能特色：

1. **多種比對模式**：

   * **內容相似度**：透過讀取檔案的 Hash (雜湊值) 及檔名相似度，精準找出內容相同或相似的檔案。
   * **副檔名及大小**：快速比對副檔名與檔案容量相同的項目。
2. **強大的預覽引擎**：

   * **多媒體**：支援一般圖片以及各種影片 (mp4, mov, avi, mkv 等) 抓取第一幀進行預覽。
   * **辦公文件**：支援 Word (doc/docx)、PDF 極速預覽。
   * **程式與純文字**：支援 txt, csv, md, json, log, cpp, h, ino 等，並具備自動偵測中文字型功能，確保中文不亂碼。
3. **優化的使用者體驗**：

   * 視窗支援拖曳調整左右比例。
   * 搜尋過程在背景多執行緒處理，不會造成視窗卡頓，並提供載入中的提示視窗。

## 使用說明

### 系統與附屬檔案需求

為了讓程式發揮針對影音檔案的最大功能，執行 `DSP4\_logEnt.exe` 時，請確保以下附屬元件與執行檔位於相同的 `DSP4\_logEnt` 資料夾中：

* `ffmpeg.exe` / `ffprobe.exe`：作為影音處理輔助元件。
*(註：v4 版本起，已不再需要 poppler 資料夾！)*

### 操作步驟

1. **開啟程式**：點擊執行 `DSP4\_logEnt.exe`。
2. **設定路徑**：在「資料夾路徑」中，使用 `Ctrl+C` 複製和 `Ctrl+V` 貼上您想要掃描的目標資料夾路徑。
3. **設定比對參數**：

   * 選擇 **「內容相似度」** 或 **「副檔名及Size」**。
   * 若為內容相似度，可自行修改 `比較內容值` (預設為 10，數值為取樣區塊大小，愈大速度愈慢)。
4. **開始搜尋**：點擊「搜尋開始」按鈕，等候系統比對檔案。
5. **預覽檔案**：

   * 搜尋結果會條列於左側窗格中（以分隔線區分不同組的重複項目）。
   * 點擊列表中的任何項目，右側窗格即可預覽檔案的內容（可滑動滾輪檢視長文字與多頁 PDF）。
6. **刪除檔案**：

   * 確認是多餘的重複檔案後，選取該行，再點擊下方的 **「刪除檔案」** 按鈕將其移除。
